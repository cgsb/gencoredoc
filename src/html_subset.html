The Format For Documentation

We use now a subset of XHTML:

  • Tags h1, h2, h3.
  • Structure p, ul, ol.
  • Links a (with attributes id and/or href).
  • Formatting i, b, tt, code.

Special spans (RFC … Not Implemented):

  • <span class="gclink" title="person(sebastien.mondet@nyu.edu)">seb</span>: Seb.
  • <code class="ocaml_code">let stuff = 42</code> and idem for <pre>
