Instructions For Building and Installing

External Dependencies

For Biocaml/Sequme/Hitscore/Hitscoreweb

See the now separated install dependencies document.

For Gencoredoc

Gencoredoc wants ocamldoc, m4, and inkscape in the $PATH.

Biocaml/Sequme/Hitscore/Hitscoreweb

All these projects now should get compiled easily with:

ocaml setup.ml -configure --prefix $OCAMLPREFIX
make
make install

Hitscore has a “make install-version” target to install an executable named with the version number: hitscore-x.y

Hitscoreweb has a “make static” target to build the native version of the website and statically link it with ocsigenserver and all the dependencies.

Gencoredoc

Call make doc in biocaml/, sequme/, and hitscore/ and then type make.

If the directory structure is not the same, set the variables HITSCORE_DOC, SEQUME_DOC, and/or BIOCAML_DOC in a file setup.data, e.g.: 

echo 'HITSCORE_DOC = ../../some/special/path/to_hiscore' > setup.data
